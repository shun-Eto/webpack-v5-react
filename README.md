# **webpack-v5-react**

## **Description**

Webpack-v.5 を用いた React の最小ディレクトリ構成

---

## **Install**

$ git clone https://github.com/shun-Eto/webpack-v5-react.git

---

## **Usage**

$ npm install

or

$ yarn install

---

## **Demo**

$ npm run start:dev

or

$ yarn start:dev

---

## **ディレクトリ構成**

- **src**

  - api

  - assets
    - images
    - modules
    - styles
    - utils
  - redux
    - store
    - slices
  - types
  - views
    - Components
    - Pages
    - Providers
    - Route
  - index.tsx
